# Java Package

Generated Protocol Buffer bindings for Java using [protobuf-javalite](https://developers.google.com/protocol-buffers/docs/reference/java-generated).

## Installation

```bash
mvn install
```

## Build

```bash
mvn compile
```

## Usage

```java
import foundation.lupyd.User;
import foundation.lupyd.FullUser;
import foundation.lupyd.auth.UserTokens;
```
