# TypeScript Package

Generated Protocol Buffer bindings for TypeScript using [ts-proto](https://github.com/stephenh/ts-proto).

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

Generated files will be placed in the `dist/` directory.

## Usage

```typescript
import { User, FullUser } from '@lupyd/protos';
import { Auth } from '@lupyd/protos/auth';
```
